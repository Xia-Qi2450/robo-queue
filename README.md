
# robo-queue
A small Python program that is useful for Robotics competition preparation
Uses the Tkinter library for GUI, Pygame for playing audio and pyttsx3 for Text-to-Speech

## Requirements

For Windows:

> pygame v2.6.1 or higher 
> pyttsx3 v2.99 or higher

For MacOS: 

> pygame v2.6.1 or higher

## Installing

Clone this Repository by:

    git clone https://github.com/Xia-Qi2450/robo-queue/

Install the requirements:

	cd robo-queue
	pip install -r requirements.txt
Run the program:

	python robo-queue-youros.py

Enjoy!
